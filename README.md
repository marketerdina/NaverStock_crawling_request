# 네이버 증권 데이터 크롤링 기반 마켓 인사이트 분석

## 1. 프로젝트 개요

본 프로젝트는 네이버의 금융 서비스인 네이버 증권 웹페이지를 `requests`와 `BeautifulSoup`을 활용해 크롤링하고, 국내 증시 지수 및 인기 종목 데이터를 수집·정제·구조화하는 실습 프로젝트입니다.

단순 데이터 수집을 넘어, 시장 흐름을 마케팅 전략 관점에서 해석할 수 있는 기반 데이터를 확보하는 것을 목표로 합니다.

---

## 2. 분석 목적 (마케팅 인사이트 관점)

금융 및 투자 데이터는 단순한 경제 지표가 아니라 소비 심리와 시장 분위기를 반영하는 선행 지표입니다.  

본 프로젝트는 다음 질문에 답하기 위해 설계되었습니다.

- 현재 시장은 상승 심리인가, 하락 심리인가?
- 투자자 관심이 집중되는 종목은 무엇인가?
- 시장 관심도 변화는 어떤 콘텐츠 및 마케팅 전략 기회로 이어질 수 있는가?
- 실시간 트렌드를 활용한 마케팅 자동화는 가능한가?

---

## 3. 수집 데이터

### 1) 주요 국내 증시 지수

- 코스피
- 코스닥
- 코스피200
- 각 지수의 현재 값
- 등락 수치
- 평균 지수 계산

활용 인사이트

- 지수 상승기에는 투자 심리 확대에 따라 금융 및 투자 콘텐츠 수요가 증가할 가능성이 높습니다.
- 지수 하락기에는 리스크 관리, 안전 자산, 장기 투자 관련 콘텐츠에 대한 관심이 높아질 수 있습니다.
- 지수 평균값은 시장 전반의 분위기를 판단하는 기초 지표로 활용할 수 있습니다.

---

### 2) 인기 검색 종목 (Top 10)

- 실시간 인기 검색 종목명
- 각 종목의 현재 가격
- 정수형 변환 및 데이터 정제
- 데이터프레임 구조화

활용 인사이트

- 검색량 기반으로 현재 투자자 관심이 집중된 종목을 파악할 수 있습니다.
- 가격 급등 종목은 이슈, 뉴스, 정책 변화 등의 신호일 수 있습니다.
- 검색 트렌드를 활용해 실시간 콘텐츠 제작 전략을 수립할 수 있습니다.
- 금융 플랫폼, 미디어, 증권 관련 서비스의 실시간 마케팅 소재로 확장 가능합니다.

---

## 4. 기술 스택

- Python  
- requests  
- BeautifulSoup  
- pandas  

---

## 5. 데이터 처리 흐름

1. requests를 활용한 HTML 요청  
2. BeautifulSoup을 통한 DOM 파싱  
3. id 및 CSS selector 기반 데이터 추출  
4. 문자열 정제 (콤마 제거, 자료형 변환)  
5. 딕셔너리 구조화  
6. pandas DataFrame 변환  
7. 평균값 계산 및 파일 저장 확장 가능  

---

## 6. 마케팅 인사이트 확장 가능성

1) 실시간 트렌드 콘텐츠 자동화  
- 인기 종목 상위 데이터 기반 자동 뉴스 요약  
- 상승률 상위 종목 기반 콘텐츠 기획  

2) 투자 심리 기반 세그먼트 전략  
- 상승장: 공격적 투자 상품 및 고위험·고수익 상품 노출 강화  
- 하락장: 안정 자산, 분산 투자, 리스크 관리 콘텐츠 강화  

3) 데이터 기반 퍼포먼스 마케팅  
- 검색 관심 종목 기반 키워드 광고 확장  
- 특정 테마 급등 시 연관 상품 프로모션 기획  

4) CRM 및 마케팅 자동화  
- 시장 상황에 따라 푸시 메시지 전략 차별화  
- 실시간 지수 변동 기반 마케팅 트리거 설계  

---

## 7. 비즈니스 관점 핵심 메시지

이 프로젝트는 단순한 크롤링 실습이 아니라,  
시장 데이터를 마케팅 의사결정 자산으로 전환하는 첫 단계입니다.

실시간 금융 데이터는  
- 소비 심리의 지표이며  
- 투자자 행동의 신호이며  
- 콘텐츠 기획의 소재이며  
- 퍼포먼스 마케팅 타겟 전략의 근거가 됩니다.  

---

## 8. 향후 고도화 방향

- 일별 데이터 누적 저장을 통한 시계열 분석  
- 상승률 및 변동성 기준 자동 랭킹 시스템 구축  
- 알림 시스템 연동 (메일, 메신저 등)  
- 대시보드 시각화 도구 연계  
- 검색량, 뉴스 데이터와 결합한 확장 분석

본 프로젝트는 웹 크롤링부터 데이터 정제, 구조화, 인사이트 도출까지의 전 과정을 실습하며,  

금융 데이터를 단순 정보가 아닌 마케팅 전략 수립을 위한 실시간 인사이트 자산으로 활용할 수 있는 기반을 마련합니다.
